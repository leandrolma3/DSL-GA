================================================================================
COMANDOS PARA EXECUTAR TESTE E SALVAR LOG NO GOOGLE DRIVE
================================================================================

PASSO 1: Conectar ao servidor
------------------------------
ssh root@right-scotia-humor-toward.trycloudflare.com


PASSO 2: Montar o Google Drive
-------------------------------
python3 -c "from google.colab import drive; drive.mount('/content/drive')"

NOTA: Vai pedir autorização. Siga as instruções na tela.


PASSO 3: Criar pasta para logs no Drive (se não existir)
---------------------------------------------------------
mkdir -p /content/drive/MyDrive/DSL-AG-Logs


PASSO 4: Executar o teste salvando log no Drive
------------------------------------------------
cd /content/DSL-AG-hybrid && nohup python compare_gbml_vs_river.py --stream RBF_Abrupt_Severe --chunks 1 --chunk-size 6000 --seed 42 > /content/drive/MyDrive/DSL-AG-Logs/test_output.log 2>&1 &


PASSO 5: Verificar se está rodando
-----------------------------------
ps aux | grep compare_gbml


PASSO 6: Monitorar o progresso em tempo real
---------------------------------------------
tail -f /content/drive/MyDrive/DSL-AG-Logs/test_output.log


PASSO 7: Para sair do monitoramento (sem parar o processo)
-----------------------------------------------------------
Pressione: Ctrl+C

ALTERNATIVA: Se o servidor NÃO for Google Colab

OPÇÃO B: Usar rclone para sincronizar com Drive
------------------------------------------------
# Instalar rclone
curl https://rclone.org/install.sh | sudo bash

# Configurar Google Drive
rclone config

# Executar teste e copiar log para Drive a cada 5 minutos
cd /content/DSL-AG-hybrid && nohup bash -c 'python compare_gbml_vs_river.py --stream RBF_Abrupt_Severe --chunks 1 --chunk-size 6000 --seed 42 > test_output.log 2>&1 & PID=$!; while kill -0 $PID 2>/dev/null; do sleep 300; rclone copy test_output.log gdrive:DSL-AG-Logs/; done' &
