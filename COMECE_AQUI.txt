Host SSH: logged-minerals-axis-infrastructure.trycloudflare.com

PASSO 1: Montar Google Drive (NO NOTEBOOK COLAB - UMA VEZ)
-----------------------------------------------------------

Abra o notebook Colab no navegador e execute:

    from google.colab import drive
    drive.mount('/content/drive')

Autorize quando solicitado. Nao feche o notebook!

===============================================================================

PASSO 2: Upload de Arquivos (NO POWERSHELL)
--------------------------------------------

Execute:

    upload_files.bat logged-minerals-axis-infrastructure.trycloudflare.com

NOTA: Voce precisara digitar a senha 'root' varias vezes durante o upload.

===============================================================================

PASSO 3: Conectar via SSH (NO POWERSHELL)
------------------------------------------

    ssh logged-minerals-axis-infrastructure.trycloudflare.com

Digite a senha: root

===============================================================================

PASSO 4: Setup do Ambiente (DENTRO DO SSH)
-------------------------------------------

    cd /root/DSL-AG-hybrid
    python3 setup_colab_remote.py

Isso vai configurar o Drive e criar os scripts necessarios.

===============================================================================

PASSO 5: Executar Experimento (DENTRO DO SSH)
----------------------------------------------

    /root/run_experiment.sh python3 compare_gbml_vs_river.py \
        --stream RBF_Abrupt_Severe \
        --chunks 3

Ou experimento de teste (1 chunk):

    /root/run_experiment.sh python3 compare_gbml_vs_river.py \
        --stream RBF_Abrupt_Severe \
        --chunks 1

===============================================================================

PASSO 6: Acessar Resultados
----------------------------

Opcao 1: Google Drive Web
    https://drive.google.com
    -> MyDrive/DSL-AG-hybrid/experiments/ssh_session_TIMESTAMP/

Opcao 2: Via SSH (ver log)
    tail -50 /content/drive/MyDrive/DSL-AG-hybrid/experiments/ssh_session_*/logs/*.log
