🎯 OBJETIVO
-----------
Executar experimentos no Google Colab VIA SSH (terminal) com salvamento
automático no Google Drive, sem abrir navegador.

===============================================================================

📋 PRÉ-REQUISITOS (UMA VEZ APENAS)
-----------------------------------

1. Notebook Colab rodando com SSH (colab_ssh_setup.ipynb)
   ✅ Você já tem: ssh logged-minerals-axis-infrastructure.trycloudflare.com

2. Google Drive montado NO NOTEBOOK (UMA VEZ):

   Abra o notebook Colab no navegador e execute:

   ```python
   from google.colab import drive
   drive.mount('/content/drive')
   ```

   Autorize quando solicitado.
   Feche o navegador (Drive fica montado).

===============================================================================

🚀 WORKFLOW COMPLETO (Windows PowerShell)
------------------------------------------

# Substitua <SEU-HOST> pelo host real do Cloudflare Tunnel

.\colab_workflow.ps1 `
    -SSHHost <SEU-HOST>.trycloudflare.com `
    -Action all `
    -Stream RBF_Abrupt_Severe `
    -Chunks 3

Isso faz:
  1. Upload de arquivos .py e .yaml
  2. Setup do Drive
  3. Execução do experimento
  4. Salvamento automático no Drive

===============================================================================

🔧 COMANDOS SEPARADOS (Windows)
--------------------------------

# 1. Upload de arquivos
.\colab_workflow.ps1 -SSHHost <SEU-HOST>.trycloudflare.com -Action sync

# 2. Setup do ambiente
.\colab_workflow.ps1 -SSHHost <SEU-HOST>.trycloudflare.com -Action setup

# 3. Executar experimento
.\colab_workflow.ps1 `
    -SSHHost <SEU-HOST>.trycloudflare.com `
    -Action run `
    -Stream RBF_Abrupt_Severe `
    -Chunks 3 `
    -Population 120 `
    -MaxGenerations 200

# 4. Download de resultados
.\colab_workflow.ps1 -SSHHost <SEU-HOST>.trycloudflare.com -Action download

📊 PARÂMETROS DO EXPERIMENTO
-----------------------------

--stream <NOME>           Stream a executar
                          Opções: RBF_Abrupt_Severe, SEA_Abrupt_Simple, etc.

--chunks <N>              Número de chunks (padrão: 3)

--chunk-size <N>          Tamanho do chunk (padrão: 6000)

--population <N>          População do GA (padrão: 120)

--max-generations <N>     Max gerações (padrão: 200)

Exemplo:
--------
.\colab_workflow.ps1 `
    -SSHHost <SEU-HOST>.trycloudflare.com `
    -Action run `
    -Stream RBF_Abrupt_Severe `
    -Chunks 5 `
    -Population 120 `
    -MaxGenerations 200

⚠️ TROUBLESHOOTING
-------------------

PROBLEMA: "Drive não está montado"
SOLUÇÃO:  Abra notebook Colab e execute:
          from google.colab import drive
          drive.mount('/content/drive')

PROBLEMA: "Permission denied" no SCP
SOLUÇÃO:  Teste SSH: ssh <SEU-HOST>.trycloudflare.com "echo OK"
          Se funcionar, tente sync novamente

PROBLEMA: "run-experiment: command not found"
SOLUÇÃO:  source ~/.bashrc
          Ou reconecte via SSH

PROBLEMA: Sessão SSH caiu, perdi tudo?
RESPOSTA: NÃO! Tudo está no Drive.
          Download: .\colab_workflow.ps1 -Action download
          Ou acesse: drive.google.com

PROBLEMA: Como monitorar em tempo real?
SOLUÇÃO:  Em outro terminal SSH:
          ssh <SEU-HOST>.trycloudflare.com
          tail -f /content/drive/.../logs/*.log

          Ou use alias:
          drive-tail
