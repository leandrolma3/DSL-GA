⚙️ PARÂMETROS DO EXPERIMENTO
----------------------------

--stream NOME               Stream a executar (ex: RBF_Abrupt_Severe)
--chunks N                  Número de chunks (padrão: 3)
--chunk-size N              Tamanho do chunk (padrão: 6000)
--population N              População do GA (padrão: 120)
--max-generations N         Máximo de gerações (padrão: 200)
--experiment-name NOME      Nome do experimento (opcional)
--log-level NIVEL           DEBUG | INFO | WARNING | ERROR

Exemplo:
--------
!python run_experiment_colab.py \
    --stream RBF_Abrupt_Severe \
    --chunks 5 \
    --chunk-size 6000 \
    --population 120 \
    --max-generations 200 \
    --experiment-name "Teste_Final_HC_Crossover" \
    --log-level INFO

PROBLEMA: Como retomar experimento interrompido?
SOLUÇÃO: 1. Encontre último checkpoint: checkpoints/chunk_X.pkl
         2. Modifique config.yaml para pular chunks já processados
         3. Execute novamente com --chunks restantes

PROBLEMA: Log não atualiza
SOLUÇÃO: import logging
         for handler in logging.root.handlers:
             handler.flush()
